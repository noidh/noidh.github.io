XImage Trainer

XImageTrainer is a free software using for preparing image data and collaborates with other DeepLearning frameworks to train the dataset. Adding boundry box for objects of thousand images will be painful without a good UI tool. XImageTrainer provides basic tool make this task more convinient and simple.

Download the lastest release

  • ximagetrainer_win_x64 (v1.0.0).zip
  • ximagetrainer_linux_x64.zip (update soon)

Current supported Deeplearning frameworks

  • Darknet and Yolo

1. Setting deeplearning engine

Go to Darknet and Yolo to obtain and build deeplearning engine by yourself.

Click [Setting] button to open Setting dialog.

Choose Darknet's execution file and setting arguments value same as above image.

2. Create new project

Click [New Project] button to open Add Project dialog.

Enter project's name and workspace's directory. After generating, a new project's structure consist of Classes, Configuration, Train Images, Test Images.

3. Add class (category) name

Right click on [Classes] and choose [Edit Class Categories] to open Edit Class dialog. Enter new class name on each row.

4. Configurate Yolo network

Right click on [Configuration] and choose [Edit Configuration] to open Configuration dialog.

5. Add Images and ROI (Region of interest)

Right click on [Train Image] or [Test Images] and choose [Add Image] to open Open File dialog. You can select a single file or multiple files to import. Images then are copied into project's Images folder. Images already exist cannot be added.

Click on Image to open it in the center image view. Choose [Rectangle] on the toolbox and draw boundary box for object.

After drawing the bounding box, a Class Selection dialog will be popupped. You need to choose the right class for this object and click [OK] to confirm.

Notice: all images must be drawed bounding box before starting the training.

6. Run the project

Click [Run] button to Build & Start training the project. After finish, the Model file will be saved into project's Backup folder.
